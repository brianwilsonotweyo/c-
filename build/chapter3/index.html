Chapter 3: C++ Variables

in c++ Variables are containers for storing data values.

In C++, there are different types of variables (defined with different keywords),

Examples of C++ variables:

  • int - this is used to store integers or whole numbers, without decimals, such as 1234
  • double - this is used to store floating point numbers, i.e those with decimals, such as 1.5 or -2.3
  • char - this is used to store single characters, like 'a' or 'B'. The Char values are always surrounded by single
  • string - this is used to store text, such as "Hello World". String values are surrounded by double quotes
  • bool - this is used to store values that are either true or false

The general rules for constructing names for variables /unique identifiers are:
  • Names can contain letters, digits and underscores
  • Names must begin with a letter or an underscore (_)
  • Names are case sensitive (myVar and myvar are different variables)
