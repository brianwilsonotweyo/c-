# Function Description
1 abs(x) Returns the absolute value of x
2 acos(x) Returns the arccosine of x, in radians
3 asin(x) Returns the arcsine of x, in radians
4 atan(x) Returns the arctangent of x, in radians
5 cbrt(x) Returns the cube root of x
6 ceil(x) Returns the value of x rounded up to its nearest integer
7 cos(x) Returns the cosine of x, in radians
8 cosh(x) Returns the hyperbolic cosine of x, in radians
9 exp(x) Returns the value of Ex
10 expm1(x) Returns ex -1
11 fabs(x) Returns the absolute value of a floating x
12 fdim(x, y) Returns the positive difference between x and y
13 floor(x) Returns the value of x rounded down to its nearest integer
14 hypot(x, y) Returns sqrt(x2 +y2) without intermediate overflow or underflow
15 fma(x, y, z) Returns x*y+z without losing precision
16 fmax(x, y) Returns the highest value of a floating x and y
17 fmin(x, y) Returns the lowest value of a floating x and y
18 fmod(x, y) Returns the floating point remainder of x/y
19 pow(x, y) Returns the value of x to the power of y
20 sin(x) Returns the sine of x (x is in radians)
21 sinh(x) Returns the hyperbolic sine of a double value
22 tan(x) Returns the tangent of an angle
23 tanh(x) Returns the hyperbolic tangent of a double value
