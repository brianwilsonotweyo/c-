Introduction

C++ is a cross-platform language meaning can be used to develop apps on any platform i.e mobile pc or embeded systems and can be used to create high-performance applications such as games and operating systems like windows.

C++ was developed by Bjarne Stroustrup, as an extension to the C language.

C++ gives programmers alot of control over system resources and memory.

C++ has been updated 3 major times recently i.e in 2011 to C++11, 2014 to C++14, and 2017 to C++17.
